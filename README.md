# Glow Blend Website

## Overview

Glow Blend is a fictional juice and smoothie brand website developed using HTML and CSS.

The website promotes fresh juices and smoothies made with natural ingredients and includes multiple pages for product browsing and company information.

---

## Website Pages

- Home
- About
- Products
- Contact
- Green Glow Juice Details
- Orange Energy Juice Details
- Berry Boost Smoothie Details
- Mango Smoothie Details
- Strawberry Smoothie Details
- Banana Smoothie Details

---

## Features

- Multi-page website design
- Product catalog with pricing information
- Individual product detail pages
- Navigation menu across all pages
- Hero image section
- Custom CSS styling
- Consistent branding and color scheme
- Responsive layout concepts

---

## Technologies Used

- HTML5
- CSS3

---

## Project Purpose

This project was created as part of a Web Design course to practice front-end web development concepts including page structure, navigation, styling, image integration, and user-friendly website design.
